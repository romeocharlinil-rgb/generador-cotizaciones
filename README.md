# Generador de Cotizaciones

App sencilla en Python con Streamlit para crear una cotizacion basica, calcular el total automaticamente y descargarla en Excel.

## Requisitos

- Python instalado
- Streamlit
- Pandas
- OpenPyXL

## Instalacion

Ejecuta este comando en la carpeta del proyecto:

```bash
pip install -r requirements.txt
```

## Ejecutar la app

Ejecuta:

```bash
streamlit run app.py
```

La app se abrira en el navegador.

## Archivos

- `app.py`
- `requirements.txt`
- `README.md`

No usa base de datos, login, pagos ni APIs externas.
